Flowerful: The Garden Color Coordinator

Flowerful is a color coordination tool for gardens built with HTML5, CSS3, and jQuery by myself and Sofia Lau-Alves in a paired programming environment (you can find more of her work here). When users input a colour into the search field, the app makes a call to the Trefle API to find plants whose flowers or leaves feature the selected color. The app then passes that data to jQuery, which appends it to the page in a minimal, accessible manner enabled by CSS3's grid and animation featuresets. Users are also able to save their favourite plants in a popout menu for later perusal. This project in particular was a personal triumph as my partner has long struggled to find new and interesting local flora in the proper colour for our garden easily and efficiently.
