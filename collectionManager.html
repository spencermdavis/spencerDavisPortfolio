Magic: The Gathering Inventory Manager

The Magic: The Gathering Collection Manager is an inventory manager for the eponymous card game. By means of the React framework, Firebase's live database feature, and integration with the Scryfall RESTful API I built a lightweight, performant, beautiful tool for the management of players' often-large card inventories. Users can input any of Magic's over 20,000 cards into the input field and see it appear instantly in their collection. The Scryfall API allows the user to see relevant information about their card at a glance, while thanks to Firebase they can then can both access and delete their cards across app instances.
